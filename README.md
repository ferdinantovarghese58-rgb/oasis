3.Temperature Converter
A simple web application to convert temperatures between Celsius, Fahrenheit, and optionally Kelvin.

Features:
- Input field for temperature
- Dropdown or radio button for unit selection
- Convert button
- Display area for result
- Input validation (accepts only numbers)

 Extra Feature:
- Conversion to/from Kelvin

---

Conversion Formulas

- Celsius → Fahrenheit:  
  F = (C × 9/5) + 32  

- Fahrenheit → Celsius:  
  C = (F − 32) × 5/9  

- Celsius → Kelvin:  
  K = C + 273.15  

- Kelvin → Celsius:  
  C = K − 273.15
